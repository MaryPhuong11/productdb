Multi-Language Product Management

Overview

This application allows you to manage products with support for multiple languages. You can add, edit, and view products in different languages simultaneously.

Features
  • Language Management: Add, view, and delete languages
  • Product Management: Add products with translations in multiple languages
  • Language Switching: View products in different languages
  • Flexible Translation: Add products with as many or as few language translations as needed
How to Use
1. Managing Languages

Click on "Manage Languages" to:

  • View all available languages
  • Add new languages (use 2-letter ISO codes like EN, FR, DE, etc.)
  • Delete languages (only if no products are using them)
2. Adding Products

You have two options for adding products:

  • Simple Form: Add products with English and Vietnamese translations only
  • Multi-Language Form: Add products with translations for all available languages
3. Viewing Products

Use the language switcher at the top of the page to view products in different languages. Only products with translations in the selected language will be displayed.

4. Searching Products

Use the search bar to find products by name or description:

  • Search works within the currently selected language
  • Search is case-insensitive and supports partial matches
  • Search both product names and descriptions
  • Use the "Clear" button to reset search results
